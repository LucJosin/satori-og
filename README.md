## Installation

### Npm

```
npm install @lucjosin/satori-og
```

### Yarn

```
yarn add @lucjosin/satori-og
```

### Cdn

```html
<script src="https://cdn.jsdelivr.net/npm/@lucjosin/satori-og/dist/index.umd.min.js"></script>
```
